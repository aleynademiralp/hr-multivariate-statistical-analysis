# Human Resources Multivariate Statistical Analysis

## Objective
This project conducts an end-to-end multivariate statistical analysis on a Kaggle Human Resources dataset. The primary goal is to extract strategic insights from complex HR data by evaluating group differences, reducing dimensionality, and building predictive classification and segmentation models.

## Tools & Technologies
* **Languages & Software:** R, SAS, SPSS
* **Dimensionality Reduction:** Principal Component Analysis (PCA), Factor Analysis
* **Statistical Testing:** One-Way MANOVA, Two-Way MANOVA
* **Predictive Modeling & Segmentation:** Discriminant Analysis, Logistic Regression, Cluster Analysis

## Methodology
1. **Structural Analysis & Hypothesis Testing:** Initiated the project with comprehensive data exploration. Applied **One-Way and Two-Way MANOVA** to statistically evaluate differences between various employee groups across multiple continuous dependent variables.
2. **Dimensionality Reduction:** Implemented **PCA and Factor Analysis** to identify underlying data structures, eliminate multicollinearity, and reduce the dataset into fewer, highly explanatory components.
3. **Predictive Modeling:** Developed classification models using **Discriminant Analysis and Logistic Regression** to predict key HR outcomes (e.g., employee turnover/attrition). 
4. **Segmentation:** Utilized **Cluster Analysis** to segment employees into distinct profiles, transforming raw HR data into actionable, strategic business insights.

## Key Results
* Successfully transformed a complex, multi-variable HR dataset into interpretable strategic insights.
* Identified the core factors driving employee behavior and grouped them into distinct segments using robust multivariate techniques across three different statistical environments (R, SAS, SPSS).
